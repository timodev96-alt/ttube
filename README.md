# ttube
##### A Simple CLI app that lets you Download Youtube Videos -or just its audio-!

## Installation
```
pip install ttube
```

# How to Use

### 1.Run
```
ttube "Video URL"
```
##### Choose File location -Optional-
```
ttube "Video Url" -o "C:\Downloads"
```

### 2.Choose A format
```
What do you want to download?
  [1] Full Video
  [2] Audio only
```

### 3.Choose The Quality!
```
Video choose a quality:
  [1]   2160p (25fps)
  [2]   1440p (25fps)
  [3]   1080p (25fps)
  [4]   720p (25fps)
  [5]   480p (25fps)
  [6]   360p (25fps)
  [7]   240p (25fps)
  [8]   144p (25fps)
```
```
Audio choose a quality:
  [1]   129 kbps (Audio)
  [2]   128 kbps (Audio)
  [3]   48 kbps (Audio)
  [4]   46 kbps (Audio)
```
